**关键证据**：
- 温彬 seedUrl "2026年稳投资政策或加码" (21jingji.com) — **2026年新文章**，HTTP 200 ✅
- 温彬 seedUrl "应对老问题与新挑战" (21jingji.com) — **2026年新文章**，HTTP 200 ✅
- 曾刚 seedUrl "十五五期间对外开放" (shifd.net) — **2025年文章**，HTTP 200 ✅
- 三个 URL 均通过 min_year=2024 过滤

**预期收益**：若扩展到 `对公贷款`，预期 raw 净增 +2~5 条（温彬/曾刚新文章），usable 池从 28 条扩大到 30~33 条。

**实施动作**：修改 `run-analyst-fetch.sh` 第48行：
```bash
# 现状
"${PYTHON}" "${FETCH_SCRIPT}" --dimension 对公存款

# 草案修改（加一行注释，保留对公存款作为默认）
# 草案：扩展维度以抓取温彬/曾刚高优来源
"${PYTHON}" "${FETCH_SCRIPT}" --dimension 对公贷款
```
或更安全的做法：

**风险**：
- 温彬/曾刚的 seedUrls 能否持续返回新文章（待验证）
- 扩展维度可能引入其他非目标来源（需检查 filter 逻辑）
