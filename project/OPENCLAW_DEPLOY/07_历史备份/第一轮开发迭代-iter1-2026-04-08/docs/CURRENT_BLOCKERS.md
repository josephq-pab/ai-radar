## BLK-001 — A Gate 贷款利率数据缺失

### 事实
- **文件：** `04_数据与规则/processed/loan_rate.json`（observedAt = 2025-12-01）
- **问题：** 最新贷款利率数据为 2025-12-01（`loan_rate.json` 的 `observedAt = 2025-12-01`）
- **目标：** `observedAt ≥ 2026-03-01`
- **缺口：** 缺少 2026-03 贷款利率 Excel 文件

### 影响范围

| 影响项 | 程度 |
|--------|------|
| A Gate 状态 | ❌ 始终 BLOCKED |
| 周报利率数据 | 只能标注 2026-02，无法标注"最新" |
| 前端利率展示 | 月份标注为 2025-12，数据已过期 4 个月 |
| 报告引用 | ⚠️ 需注明数据月份，不得声称"截至 2026-03" |

### 临时绕行方式

1. **周报/前端标注法**（立即可用）：
   - 将利率数据标注为 `(2025-12)` 或 `(截至 2025-12)`
   - 在引用处注明"数据待更新至 2026-03"
   - 禁止标注为"最新"或"截至 2026-03"

2. **报告输出限制**：
   - 定价观察一节需加注：`利率数据截至 2025-12，请结合最新数据校正`

- 需要 2026-03 月度的贷款利率 Excel（股份行人行口径）
