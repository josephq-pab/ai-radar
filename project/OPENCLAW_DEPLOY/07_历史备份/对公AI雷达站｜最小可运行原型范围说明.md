# 对公AI雷达站｜最小可运行原型范围说明

## 1. 原型目标
以最小范围验证“首页总览—双案例承接—重点事项闭环—周报/专题入口联动”这条链路是否可跑通，不进入高保真视觉层，不扩展新页面体系。

## 2. 最小原型范围
### 页面
1. 首页总览
2. 存款深案例页
3. 贷款深案例页
4. 重点事项跟踪页

### 能力
1. 信息汇总与重点展示
2. 存款/贷款双案例承接
3. 跟踪事项闭环
4. 周报/专题入口联动

## 3. 首期必须具备
- 4 个核心页面可访问
- 页面间主跳转链路可用
- 存款/贷款样板内容可承接
- 跟踪页可展示首批事项
- 对待补数据、待补口径有明确标识

## 4. 首期暂不纳入
- 高保真视觉稿
- 复杂权限体系
- 审批流/自动派单
- 自动专题生成引擎
- 全量风险模型接入

## 5. 原型完成判断标准
- 首页、存款页、贷款页、跟踪页已能串联
- 双案例可稳定承接样板内容
- 跟踪页可承接事项状态与下一步动作
- 周报/专题入口具备最小承接能力
- 页面、事项、周报三者基本一致
