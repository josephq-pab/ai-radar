## 一、本轮定位

**本轮不是**：继续收集结构性反馈 / 最终定稿 / 正式部署

**本轮是**：V6 使用黏性验证轮——用 V6 做第二轮真实试发，重点验证"顺手度是否因轻量收窄而提升"和"持续接收意愿是否从 B 转化为 A"。

---

## 二、与首轮试发（TRL-01）的差异

| 维度 | 首轮（TRL-01）| 二轮（TRL-02）|
|------|--------------|---------------|
| 核心问题 | "结构对不对？" | "顺手度是否提升？" |
| 问卷长度 | 领导版3题 + 执行版4题 | 统一4题（精简）|
| 问卷结构 | 问"结构是否正确" | 问"是否比上一版更好" |
| 首轮已问问题 | Q1（企金归口）/ Q2（双门槛）/ Q3（三档）/ Q4（持续意愿）| **全部不再重复问** |
| 二轮新问 | — | 顺手度自评 / 动作可用性自评 / 篇幅合适度 |
| 对象范围 | 6人（4成功+2失败）| 核心3-4人（减法，控质量）|

| 编号 | 问题 | 类型 | 首轮对应 |

- Q2（双门槛结构）：0/6 反对 → 直接进入推荐，不问

## 五、成功门槛

| 门槛 | 标准 | 对应验证点 |
|------|------|---------|
| T2-T1 | T2-Q1 有≥1人回答"明显更容易懂"或"稍微容易懂" | 顺手度提升 |
| T2-T2 | T2-Q2 有≥1人回答"更像"或"有点像" | 动作可用性 |
| T2-T3 | T2-Q3 有≥1人回答"愿意"（A） | 持续意愿从B→A |
| T2-T4 | T2-Q4 篇幅合适（A或B） | 篇幅合适度验证 |

**进入"更正式试用周期"的最低条件**：T2-T1 + T2-T2 + T2-T3 全部满足

---

## 六、判断标准

| 条件 | 结果 |
|------|------|
| T2-T1 + T2-T2 + T2-T3 全部满足 | ✅ V6顺手度验证通过，可进入更正式试用周期 |
| T2-T1 + T2-T2 满足，但 T2-T3 不满足 | ⚠️ 顺手度提升但黏性不足，需V7微调 |
| T2-T1 或 T2-T2 任一不满足 | ⚠️ 轻量收窄方向存疑，需针对性调整 |
| 出现明确反对（选C比例>50%）| ❌ V6方向有误，需回退 |
