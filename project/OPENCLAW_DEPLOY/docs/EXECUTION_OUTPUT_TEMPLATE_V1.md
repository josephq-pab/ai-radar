## 一、版式定位

- **阅读对象**：一线业务人员、部门执行层
- **阅读时间**：3~4 分钟（V0 为 3~5 分钟，压缩后）
- **核心要求**：动作清晰，知道先做什么，知道怎么看效果

## 三、V1 与 V0 的执行版对比

| 维度 | V0（执行版）| V1（执行版）|
|------|------------|------------|
| 主动作数量 | 3条混合 | 2条立即执行 + 1条研究准备（分类） |

## 四、执行版动作分类标准（V1 核心规则）

### 4.1 立即执行动作（"二"）

**定义**：本月必须有人推动，完成后能直接影响业务结果。

**判断标准**：执行人拿着这条动作，能在月底前完成一件具体的事（联系客户/发出指令/完成排查/发出报告）。

**不允许出现**：
- "获取数据"类（这类是研究准备，不是立即执行）
- "建立机制"类（机制建好是下个月的事，不是本月执行）
- "做分析"类（分析是研究准备，不是执行）

### 4.2 研究准备动作（"三"）

**定义**：本月需要有人去搞清楚一件事，结果给下个月提供判断依据。

**判断标准**：执行人拿到这条动作，本月做完后，只能产出"结论"而不是"业务动作"。

**格式要求**：
- 必须写明"预计产出"：下月初可形成什么结论
- 必须写明"此动作本月不产生直接业务动作"
- 不得占用"主动作"名额

**不允许出现**：
- 把研究准备写成主动作（"搞清楚"不等于"做到了"）
- 研究准备动议的验证指标写成业务结果（研究结果≠业务结果）

### 4.3 观察项（"四"）

**定义**：值得监测但本月不需采取行动，信号出现后才升级。

**格式要求**：
- 必须写"执行人/部门"
- 必须写"预警信号"（量化）

## 五、主动作"研究准备"与"立即执行"的边界说明

**典型错误（V0 样稿）**：
> 主动作2：获取华夏银行存款利率截面数据，判断增速持续性

这条是研究准备，不是立即执行。执行层拿到"利率截面数据"以后，还是不知道怎么办——要等分析完才有结论。

**正确处理（V1）**：
- 放在"三、研究准备"
- 结论是给下个月用的（"下月判断华夏增速是否趋势性"）
- 本月不产生直接业务动作

## 六、版本变更说明（V0 → V1）

| 变更点 | V0 | V1 | 变更原因 |
|--------|----|----|---------|
| 主动作数量 | 3条（含研究动作混入）| 2条立即执行+1条研究准备 | 解决"执行层拿到不知道怎么动" |
| 动作类型分类 | 无 | 明确区分"立即执行/研究准备" | 核心改变 |
| 观察项执行人 | 无 | 必须写 | 解决"每月看了但没人负责" |
| 贷款侧表达 | 表格形式 | 文字叙述+关键数字 | 更像业务表达 |
| 两版独立性 | 降调项重复 | 各写各的 | 不再复制粘贴 |
| 动作结论句 | 问句（"是否有异动"）| 判断句（"立即排查"）| 更像执行指令 |
