## 二、V4 四段结构

```
━━━ 【报告标题】━━━━━━━━━━━━━━━━━━
月度同业存款/贷款分析
（数据截至 XXXX年XX月）

━━━ 【一、本月最重要变化】（≤3条）━━━━━━━━━━
[四字判断 + 支撑数据 + 一句话意义]

1. 【中信存款攻势超预期】
    中信银行当月新增存款2401亿，增量居样本行第一，超出预期。
    [背景：两会后项目融资落地，支撑其对公存款增长]

2. 【华夏存款增速领先但待验证】
    华夏银行年增+6.3%，增速全行业最高。
    [但样本期仅3个月，尚无法判断是趋势还是季节性波动]

3. 【平安外币存款流入加快】
    平安银行外币存款月增+121亿，年增+19.7%，外币资金流入提速。

━━━ 【二、原因判断与对我行意义】━━━━━━━━━

针对变化1（中信攻势）：
    机制：两会后稳投资政策落地，对公结算存款同步增加
    置信度：【中】逻辑通顺，但缺中信贷款投向直接数据
    对我行意义：中信对公存款攻势若持续，可能分流我行重点客户存款

针对变化2（华夏提速）：
    机制：定期储蓄产品利率竞争力较强，叠加网点扩张
    置信度：【低】样本期仅3个月，尚无法判断是趋势还是季节性波动
    对我行意义：华夏存款定价策略持续性待观察，下月增速是否回落是关键信号

针对变化3（平安外币流入）：
    机制：美联储降息预期下，客户锁汇意愿下降，部分资金从理财回流本币
    置信度：【低】样本期短，需持续跟踪
    对我行意义：外币存款的分流效应可能同步影响我行外币业务

━━━━━━━ 【三、三条建议】━━━━━━━━━━

1. 【立即排查中信挖转风险】
    目的：确认中信存款增量中是否有来自我行的转移存款
    建议责任人：企金部门（注：企金部/战略研究部均可承接，视各行实际）[当前推荐·强确认]
    验证指标：
        重点排查：联系5000万以上存款到期客户，确认续做意向
        快速扫描：排查1000万以上存款客户有无异常信号（重点是有无异常，不必全部联系）
        预警阈值：存量客户中单笔转投金额≥1亿元，上报分行公司部负责人

2. 【监测华夏定价策略走向】
    目的：判断华夏存款增长是价格驱动还是战略扩张
    建议责任人：企金部门（注：企金部/战略研究部均可承接，视各行实际）[当前推荐·强确认]

3. 【启动招行贷存款联动预警】
    目的：招行贷款利率2.54%已是行业最低，需提前预警
    建议责任人：分行公司业务部门

━━━━━━━ 【四、两个观察项 ━━━━━━━━━━━━

▷ 观察项A：外币存款流入是否持续
    频率：每月
    预警：若连续2个月>100亿且我行外币存款流失>10%，升级为主动作

▷ 观察项B：定期储蓄存款占比是否加速上升
    频率：每月
    预警：样本行中3家以上月增占比>2个百分点，升级为主动作
```
