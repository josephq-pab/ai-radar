## 版本变更说明（V5→V6）

| 变更点 | V5 | V6 | 变更原因 |
|--------|----|----|---------|
| 三档条件表达 | 2行注语 | **V4/V5 完整段落**（恢复数值背景+三出口）| Q2=B：逻辑未被否定，但压缩后指导性不足 |
| 主动作格式 | 动作名+目的+做法 | **动词打头+具体操作标准** | Q3=C：动作不像能部署，分析报告感太强 |
| "怎么做"层 | 偏分析描述 | **可部署的操作标准**（具体动作/话术/判断标准）| Q1=D：执行者拿到的是摘要不是行动工具 |
| 研究准备"研究内容" | 3项选2项，无完成标准 | **明确完成标准和产出格式** | Q2=B：压缩过头，执行者不知如何算完成 |
| 执行版篇幅 | 约 850 字 | **约 950~1000 字** | 必要信息厚度回归，不是 V5 冗余回归 |
| 其余结构 | 不变 | 不变 | 企金归口/双门槛/三段式不变 |

## 一、版式定位

- **阅读对象**：一线业务人员、部门执行层
- **阅读时间**：3~5 分钟
- **核心要求**：动作清晰，知道先做什么，知道怎么看效果，少读多懂

## 二、V6 执行版段落结构

```
━━━ 【执行版】月度同业存款/贷款分析行动指引
（数据截至 XXXX年XX月）

━━━ 【一、本月存款事实】

当月中信新增存款2401亿，增量居样本行第一。
华夏年增+6.3%（全行业最高），但月增仅+558亿，规模仍小。
平安外币存款月增+121亿，外币资金流入提速。

━━━━━━━ 【二、主动作：立即执行】━━━━━━━━━━

▶ 主动作1（本月必做 / XX月XX日前）：排查中信挖转风险

  做这件事是为了：确认中信存款增量是否来自我行转移
  建议责任人：企金部门（注：企金部/战略研究部均可承接，视各行实际）[6源一致确认] / 客户经理

  【具体怎么做】
  第一步：筛选存量客户中当月存款到期客户（5000万以上），逐一联系，确认续做意向
    → 若客户明确表示"有他行高息存款计划"，记录并立即上报分行公司部负责人
  第二步：排查1000万以上客户有无异常信号（不需要全部逐一联系，重点扫有没有异常）
    → 若发现存量客户有单笔转投≥1亿元迹象，立即上报分行公司部负责人
  第三步：整理一份"需重点关注的存量客户"名单（≥5000万），提交分行公司部负责人

  参考阈值：5000万/1000万为参考值，各行可按存量客户数量和执行承载力上下浮动
  完成时点：XXXX年XX月XX日前

▶ 主动作2（本月必做 / XX月XX日前）：完成一次招行贷存款联动客户扫描

  做这件事是为了：招行贷款利率2.54%已是行业最低，本月建基准数据，判断我行优质客户是否正在被招行接触
  建议责任人：分行公司部 / 客户经理（主办）+ 产品经理（支持）

  【具体怎么做】
  第一步：联系年结算量>1亿的优质客户，直接问："近期有没有招行的人联系过您？"
    → 若3户以上重点客户反映"有招行联系过"，立即上报分行公司部负责人
    → 若有客户明确表示"正在比较我行和招行的利率"，记录并上报
  第二步：整理"招行正在接触的客户"名单，提交分行公司部负责人和产品经理
  完成时点：XXXX年XX月XX日前

━━━━━━━ 【三、主动作：研究准备】━━━━━━━━━━

▸ 研究准备（XX月XX日前）：华夏存款增速趋势判断

  搞清楚这个问题是为了：为下月判断"华夏存款增速是否为趋势性变化"提供依据
  研究责任人：企金部门（注：企金部/战略研究部均可承接，视各行实际）[6源一致确认]
  接收方：分行公司部负责人（或分行长）

  研究内容（3项选2项即可，不用全做）：
    1. 华夏3年/5年期定存利率与我行差值（判断是否价格驱动）
       完成标准：拿出一张对比表，写明"华夏比我行高多少bp"
       产出格式：3~5行文字+1张利率对比表
    2. 近三月月均新增物理网点数量（判断是否渠道扩张驱动）
       完成标准：拿到近三月网点净增数量
       产出格式：3行数字
    3. 近3年Q1同比增速历史对比（判断是否季节性）
       完成标准：拿到近3年Q1同比增速数据
       产出格式：3行数字

  产出形式：一页书面判断报告（3~5个要点）【打头第一条结论必须最核心】
  接收方：分行公司部负责人（或分行长）

  转动作条件（V4/V5 完整段落格式，恢复数值背景）：
    【条件A — 触发动作】
    若同时满足以下两个条件，则下月启动针对华夏的存款挖转排查：
      (1) 华夏3年定存利率高于我行15bp以上（价格驱动强）
      (2) 华夏近三月月均新网点>5个（渠道扩张驱动明显）
    → 下月启动挖转排查：重点联系有华夏网点覆盖区域的存量客户

    【条件B — 归档】
    若华夏月增连续2个月<500亿，说明增速回落后不再构成主要威胁
    → 下月归档，不再触发针对华夏的存款动作

    【条件C — 待定】
    其余情况（满足价格驱动但无渠道扩张；或满足渠道扩张但价格驱动不足）
    → 结论留存，下月补充数据后再判断

  预计完成时点：XXXX年XX月XX日前

━━━━━━━ 【四、观察项】━━━━━━━━━━

▷ 观察项A：外币存款流入是否持续
    执行人/部门：分行公司部 / 客户经理（日常工作中自然承担）
    关注：连续2个月>100亿 且 我行同期外币存款流失>10% → 升级为主动作

▷ 观察项B：定期存款占比是否加速上升
    执行人/部门：企金部门（注：若无专门研究支持，可由企金部产品经理兼任）[6源一致确认]
    关注：样本行中3家以上月增占比>2个百分点 → 升级为主动作
```

## 三、三档条件完整段落格式（V6 恢复）

V6 将 V5 的 2行注语回退到 V4/V5 完整段落格式，恢复数值背景和三出口：

**V5（2行注语，问题：缺数值背景，执行者无法判断）**：
```
转动作条件（注语形式，压缩版）：
    · 若华夏3年定存利率高于我行15bp以上 且 月均新网点>5个 → 下月启动挖转排查
    · 若华夏月增连续2个月<500亿 → 下月归档
    · 其余情况：结论留存，下月补充数据后再判断
```

**V6（完整段落，恢复数值背景和三出口）**：
```
转动作条件（完整段落）：
    【条件A — 触发动作】
    若同时满足以下两个条件，则下月启动针对华夏的存款挖转排查：
      (1) 华夏3年定存利率高于我行15bp以上（价格驱动强）
      (2) 华夏近三月月均新网点>5个（渠道扩张驱动明显）
    → 下月启动挖转排查：重点联系有华夏网点覆盖区域的存量客户

    【条件B — 归档】
    若华夏月增连续2个月<500亿，说明增速回落后不再构成主要威胁
    → 下月归档，不再触发针对华夏的存款动作

    【条件C — 待定】
    其余情况（满足价格驱动但无渠道扩张；或满足渠道扩张但价格驱动不足）
    → 结论留存，下月补充数据后再判断
```

**为什么回退**：
- 2行注语缺乏数值来源说明（为什么是15bp？为什么是5个网点？）
- 执行者拿到结论无法自主判断阈值是否合理
- 完整段落恢复数值背景，执行者可理解、可验证、可调整
