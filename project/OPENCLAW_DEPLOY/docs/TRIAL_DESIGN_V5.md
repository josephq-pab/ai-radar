## 一、试发目的

**不是为了验证"V5已经足够好"，而是为了回答：V5是否值得进入"更正式的试用周期"。**

---

## 二、试发对象建议（最小范围：3人）

| 对象 | 类型 | 验证优先级 | 说明 |
|------|------|-----------|------|
| 对象A | **领导视角** | L-1（结构鲜明度）、L-2（篇幅合适度）| 优先，已验证L-1/L-2从未被真正验证 |
| 对象B | **执行负责人** | P1/P2/P3（值不值得看/看不看得懂/动作能不能用）| 执行版核心验证 |
| 对象C | **潜在持续试用** | P4（愿不愿继续收）| FB-01/02/03已选B，此为辅助验证 |

**试发对象选择原则：**
- 不选已参与FB-01/02/03填答的人（避免重复验证同一来源）
- 优先选从未参与过AI雷达站问卷填答的人
- 对象A优先于对象B/C（L-1/L-2从未被验证）

## 三、试发包（V5 试发包）

### 3.1 领导版试发包

**发送内容**：领导版V4样稿（月度分析报告领导版格式）

**想验证的问题：**
1. L-1：是否一眼看出本月最重要变化是哪个？（A/B/C）
2. L-2：篇幅（≤1300字）是否合适？（A/B/C）
3. 看完后，是否觉得下月还想继续收？（A/B/C）

**发送格式**：
```
请查阅附件《月度同业存款/贷款分析》领导版草稿（V5试用版）。

收到后请回答：
1. 您是否能一眼看出本月最重要变化？（A.能 B.大概能 C.不能）
2. 篇幅（≤1300字）是否合适？（A.合适 B.可以再短 C.偏长）
3. 下月是否愿意继续收到这样一份分析报告草稿？（A.愿意 B.可以试试 C.不愿意）

可直接回复此邮件。
```

### 3.2 执行版试发包

**发送内容**：执行版V4样稿（月度分析报告执行版格式）

**想验证的问题：**
1. P1：看完主动作，是否觉得内容值得花时间看？（A/B/C）
2. P2：看完主动作，是否知道本月要做什么？（A.完全知道 B.大概知道 C.不知道）
3. P3：验证指标是否可操作？（A.可操作 B.部分可操作 C.不可操作）
4. P4：下月是否愿意继续收？（A/B/C）

**发送格式**：
```
请查阅附件《月度同业存款/贷款分析》执行版行动指引（V5试用版）。

收到后请回答：
1. 看完主动作，是否觉得内容值得花时间看？（A.值得 B.不一定 C.不值得）
2. 看完主动作，是否知道本月要做什么？（A.完全知道 B.大概知道 C.不知道）
3. 验证指标是否可操作？（A.可操作 B.部分可操作 C.不可操作）
4. 下月是否愿意继续收到这样一份分析报告草稿？（A.愿意 B.可以试试 C.不愿意）

可直接回复此邮件。
```

## 四、试发后判断标准

### 4.1 试发成功（具备进入"更正式试用周期"条件）

满足以下条件，可判断V5具备进入更正式试用周期：
- L-1 或 P1 有≥1人回答正向（A）
- L-2 或 P2 有≥1人回答正向（A/B）
- P3 有≥1人回答正向（A/B）

### 4.2 试发部分成功（需要调整后再试）

满足以下条件，需要针对性调整后再次试发：
- L-1/L-2 均回答C（结构不鲜明/篇幅不合适），但P2/P3仍正向
- P1 回答C，但P2/P3仍正向

### 4.3 试发失败（需要重新评估）

满足以下条件，需要重新评估V5方向：
- P2 有≥1人回答C（不知道怎么做）
- P3 有≥1人回答C（验证指标不可操作）
