## 背景

P3-28 已完成辅助页单跳规则，定义了四个辅助页各自承接的问题类型。

但仍存在问题：
- 跳进辅助页后，不知道什么时候可以停止
- 不知道什么时候该回到运营页
- 不知道什么时候才允许第二跳
- 容易从单跳变成无序连跳

本功能包定义辅助页三种出口规则和二跳约束。

---

## 辅助页三种出口

| 出口类型 | 含义 | 触发条件 |
|---------|------|---------|
| ✅ 直接停止 | 问题已回答，不需要继续跳 | 当前页已完整回答了用户的具体问题 |
| ↩️ 返回运营页 | 带着结论回运营页收口 | 问题已局部解决，需要把结论带回整体状态确认 |
| 🔶 有条件二跳 | 仅当问题自然转化为另一类时才跳 | 当前页无法独立回答且新问题属于另一辅助页职责 |

---

## 各辅助页默认出口与二跳规则

### ops-evidence.html（证据页）

| 属性 | 内容 |
|------|------|
| 默认出口 | ↩️ 返回 single-chain-ops（运营页）|
| 允许二跳 | 若看完证据后引出"这算不算异常"/"要不要触发"→ 再去 ops-decision.html |
| 不建议去向 | 直接去 config-status.html / ops-playbook.html（配置/执行问题不在证据页解决）|

### ops-decision.html（决策助手）

| 属性 | 内容 |
|------|------|
| 默认出口 | ↩️ 返回 single-chain-ops（运营页）|
| 允许二跳 | 若决策结论是"应进入执行"→ 再去 ops-playbook.html |
| 不建议去向 | 直接去 ops-evidence.html / config-status.html（判断问题在决策页已解决）|

### ops-playbook.html（执行流程）

| 属性 | 内容 |
|------|------|
| 默认出口 | ✅ 停止 或 ↩️ 返回 single-chain-ops（运营页）|
| 允许二跳 | 若执行前需核对配置/Gate/参数→ 再去 config-status.html |
| 不建议去向 | 执行完毕后直接去 ops-evidence.html / ops-decision.html（执行问题在执行页已解决）|

### config-status.html（配置入口）

| 属性 | 内容 |
|------|------|
| 默认出口 | ✅ 停止 或 ↩️ 返回 single-chain-ops（运营页）|
| 允许二跳 | 若核参后仍需判断是否触发→ 再去 ops-decision.html |
| 不建议去向 | 配置核对后直接去 ops-evidence.html / ops-playbook.html（配置问题在配置页已解决）|

---

## 允许的二跳组合

| 第一跳 | 允许的第二跳 | 条件 |
|-------|------------|------|
| 证据页 → | 决策页 | 若引出"是否触发/是否异常"问题 |
| 决策页 → | 执行页 | 若结论是"应进入执行" |
| 执行页 → | 配置页 | 若执行前需核参 |
| 配置页 → | 决策页 | 若核参后仍需判断触发 |

**其余二跳组合均不推荐。**

---

## 不建议的二跳组合

| 组合 | 不推荐原因 |
|------|---------|
| 证据页 → 配置页 | 配置问题直接去配置页即可，不需要绕道证据页 |
| 决策页 → 证据页 | 判断问题在决策页已解决，不需要绕道证据页 |
| 执行页 → 证据页 | 执行问题在执行页已解决，不需要去证据页回溯 |
| 配置页 → 证据页 | 配置问题在配置页已解决，不需要去证据页 |

---

## 不建议的深查方式

| 错误做法 | 正确做法 |
|---------|---------|
| 进了辅助页就继续跳第二个 | 先问：当前页的问题回答了吗？ |
| 单跳后默认再跳一个 | 找到答案即停止，不强制二跳 |
| 把四个辅助页都跳一遍 | 带着具体问题单跳一个，不无序深查 |
| 不确定该跳哪个，先跳证据页 | 先判断问题类型再决定跳哪个 |
| 看完配置后直接去证据页 | 核对配置后默认停止或回运营页 |

---

## 当前推荐的最小深查闭环

```
核心三页闭环（首页→快报→运营）
  ↓
带着具体问题单跳一个辅助页
  ↓
找到答案
  ↓
① 问题已回答 → 停止
② 需要带回整体理解 → 返回运营页收口
③ 问题自然转化为另一类 → 有条件二跳（仅限4种允许组合）
  ↓
到达答案即停止，不连续多跳
```

---

## 当前不应误解的事项

| 误解 | 正解 |
|------|------|
| 进了辅助页应该继续多看几页 | 否，找到答案即停止，不强制多跳 |
| 单跳后默认应该再跳一个 | 否，单跳优先，非必要不做二跳 |
| 四个辅助页之间默认串联 | 否，有明确出口规则，不是串联关系 |
| 看完证据页后应该去配置页看看 | 否，证据页后不建议直接去配置页 |
| 看完配置页后应该去证据页确认 | 否，配置页后默认停止，不绕道 |
| 决策页后应该去证据页找依据 | 否，判断在决策页已解决，不绕道 |
