## 一、术语页定位

**术语页是什么：**
- 不是帮助中心
- 不是完整文档库
- 不是后台
- 是：**高频术语统一口径入口页**（ops-glossary.html）

**术语页解决什么问题：**
- 运营者对关键术语理解成本高（冻结/触发/单链路/基线等）
- 六页对同一状态的表述不完全一致
- 运营者容易误解"冻结"/"等待"/"停止"等词的含义

## 四、常见误解关系澄清

| 误解 | 实际 |
|------|------|
| RUN-01完成 ≠ 可以跑RUN-next | 必须满足触发条件才能进入下一轮 |
| 第二链路冻结 ≠ 项目停止 | 第一链路正常运行，冻结是主动选择 |
| V7基线冻结 ≠ 内容不可查看 | 冻结的是"改版动作"，不是"阅读使用" |
| 只读入口 ≠ 配置真源 | 页面是快照展示，不是配置编辑器 |
| 等待触发 ≠ 项目卡住 | 等待是Phase 3正常节奏 |
| usable充足 ≠ 可以不触发 | 池充足但RUN-next仍必须满足触发条件 |
