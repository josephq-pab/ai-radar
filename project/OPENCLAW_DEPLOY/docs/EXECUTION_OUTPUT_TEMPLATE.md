---

## 一、版式定位

- **阅读对象**：一线业务人员、部门执行层
- **阅读时间**：3~5 分钟
- **核心要求**：动作清晰、可执行、有验证指标
- **与领导版关系**：共享底层判断和证据，执行版不写推断，只写"所以要关注什么"

---

## 二、执行版标准段落结构

```
━━━ 【执行版】月度同业存款/贷款分析行动指引
（数据截至 XXXX年XX月）

━━━ 【一、本月存款现状】（事实段）━━━━━━━━━
[数据说话，存款为主；不需要推断，只需要事实]

当月存款余额情况（XX银行）：
    [格式：银行名 | 当月新增 | 年增 | 增速排名]
    - 招行：+XXXX亿，年增XX%，[排名]
    - 中信：+XXXX亿，年增XX%，[排名]
    - [以此类推]

当月贷款情况（辅助参考）：
    [极简：只说一个关键变化，如"贷款利率较上月-Xbp"等]
    [如无与存款联动的内容，则标注"贷款侧本月无显著联动变化"]

━━━━━━━ 【二、三条主动作】（行动核心段）━━━━━━━━━━

[格式：主动作编号 + 动作名称 + 目的 + 建议责任人 + 验证指标]

▶ 主动作1：[动作名称]
    目的：[做这件事是为了防范什么风险或抓住什么机会]
    建议责任人：[部门/团队级，不写具体人名]
    验证指标/观察点：
        [具体可量化或可观察的指标]
        [如：X银行对我行的存款挖转率超过X%则预警]

▶ 主动作2：[动作名称]
    目的：[同上]
    建议责任人：[同上]
    验证指标/观察点：[同上]

▶ 主动作3：[动作名称]
    目的：[同上]
    建议责任人：[同上]
    验证指标/观察点：[同上]

━━━━━━━ 【三、两条观察项】（睁眼段）━━━━━━━━━━

[不是立即要做的，而是"睁大眼睛看着"的信号]

▷ 观察项A：[观察信号名称]
    观察什么：[具体观察哪个指标或行为]
    观察频率：[每周/每两周/每月]
    预警信号：[达到什么情况需要上报或升级]

▷ 观察项B：[同上]

━━━━━━━ 【四、一个待核实项】（降调确认段）━━━━━━━━━━

需要补充的信息：
    1. [还需要什么数据/事件/观察才能形成结论]

```

## 二bis、执行版补充说明（存款为主贷款辅助落地点）

**贷款辅助内容在执行版中的处理**：
- 存款现状段（"一"）是主体，贷款侧数据如果与存款策略有关，**在"一"段末尾用一句话提及**
- 贷款侧如果有独立重要的变化（如某银行贷款利率大幅下调影响存款竞争力），**单独在"主动作"中提一条**，但标注"贷款侧动作"
- 贷款侧如果没有与存款直接相关的内容，**不在执行版中单独成段**

**示例（贷款辅助插入）**：
```
当月贷款情况（辅助参考）：
    招行最新贷款利率较上月-8bp至2.54%，已降至全行业最低区间。
    [插入点：若此利率优势持续，可能对招行一般性存款形成争夺，注意监测]
    光大银行利率上调+12bp至3.22%，价量策略出现分化。
```

## 三、写作规范

### 3.1 现状原则（"一"段）

- **只写事实**：不写推断、不写原因、不写启示，就写数据
- **存款为主**：至少6家银行的数字，贷款侧一句话带过
- **格式固定**：用表格或列表格式，方便对比，不需要阅读理解

### 3.2 主动作原则（"二"段）

- **有目的**：先写"为什么做"，再写"做什么"，防止动作空心化
- **有责任人**：建议到部门/团队级别，不写具体人名（人名是派工单的事，不是分析稿的事）
- **有验证指标**：每条必须可量化或可观察；无法量化时，写"是否发生"类观察指标
- **禁止写战略方向**：不写"加强客户黏性"这种方向，要写"本周内完成XX客户的大额存单到期提醒"
- **不写成任务单**：格式是"建议做什么"，不是"请于XX日前完成XX"

### 3.3 观察项原则（"三"段）

- **区别于主动作**：主动作是"立即要做"，观察项是"睁大眼睛看着"
- **有触发信号**：必须写出"出现什么情况需要升级"
- **有观察频率**：防止"先看看"变成"永远不看"

### 3.4 待核实原则（"四"段）

- **继承自领导版**：执行版不自己做降调判断，降调结论从领导版"六"继承
- **明确"需要什么"**：执行版关注"需要补充什么信息"，而不是"这个判断对不对"

## 四、三条主动作标准格式示例

```
▶ 主动作1：监测招行存款挖转动向
    目的：招行当月存款新增1540亿，增速最快，需关注是否针对我行重点客户开展攻势
    建议责任人：分行/客户管理部门
    验证指标：
        - 直接指标：我行重点客户群（大额存单到期客户）有多少转投招行
        - 间接指标：招行同期大额存单销量环比是否大幅增长
        - 预警阈值：存量客户中单笔转投金额超过XXX万则预警
        - 观察频率：每周

▶ 主动作2：加强华夏银行增速领先原因分析
    目的：华夏银行年增6.3%，增速全行业最高，但其绝对规模仍小，需判断是否可持续
    建议责任人：战略规划/研究部门
    验证指标：
        - 华夏银行存款产品结构（是否靠高利率产品驱动）
        - 下月华夏银行增速是否回落

▶ 主动作3：摸底我行大额存单到期分布
    目的：为下月存款保卫战做准备，提前识别流失风险客户
    建议责任人：零售/客户管理部门
    验证指标：
        - 本月内完成XXX万元以上大额存单到期客户的逐一联系
        - 到期金额中已确认续做的比例（目标≥70%）
```

## 五、长度控制

| 段落 | 建议字数 | 上限 |
|------|---------|------|
| 一、本月存款现状 | 200~300字 | 400字 |
| 二、三条主动作 | 300~400字 | 500字 |
| 三、两条观察项 | 150~200字 | 250字 |
| 四、一个待核实项 | 80~100字 | 150字 |
| **全文合计** | **≤1100字** | **1200字** |

---

## 六、与领导版的差异对照

| 维度 | 领导版 | 执行版 |
|------|--------|--------|
| 开头 | 先给结论（变化最重要）| 先给数据（现状是什么）|
| 中段 | 机制推断 | 不写推断 |
| 建议 | 含目的+责任人+验证指标 | 含目的+责任人+验证指标，更具体 |
| 观察项 | 两条，月度频率 | 两条，可更高频（周/双周）|
| 降调项 | 一个，必须有 | 继承自领导版，执行版重在"需补什么" |
| 贷款内容 | 在"三"段联动 | 在"一"段末尾或"主动作"中提一条 |
| 推断/判断 | 有（推断段+降调段）| 无推断，无降调判断 |
