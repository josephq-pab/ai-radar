```

━━━ 【一、本月存款事实】（关键数字加粗）

当月中信新增存款2401亿，增量居样本行第一。
华夏年增+6.3%（全行业最高），但月增仅+558亿，规模仍小。
平安外币存款月增+121亿，外币资金流入提速。

━━━━━━━ 【二、主动作：立即执行】━━━━━━━━━━

▶ 主动作1（本月必做 / XX月XX日前）：排查中信挖转风险

- 做这件事是为了：确认中信存款增量是否来自我行转移
- 建议责任人：企金部门（注：企金部/战略研究部均可承接，视各行实际）[6源一致确认] / 客户经理
- 怎么做：
    · 重点：联系5000万以上存款到期客户，确认续做意向
    · 扫描：排查1000万以上客户有无异常信号（不必全部联系）
    · 预警：存量客户单笔转投≥1亿元，立即上报分行公司部负责人
- 参考阈值：5000万/1000万为参考值，各行可按实际客户数量和执行承载力上下浮动
- 完成时点：XXXX年XX月XX日前

▶ 主动作2（本月必做 / XX月XX日前）：完成一次招行贷存款联动客户扫描

- 做这件事是为了：招行贷款利率2.54%已是行业最低，本月建基准数据
- 建议责任人：分行公司部 / 客户经理（主办）+ 产品经理（支持）
- 怎么做：
    · 直接问：联系年结算量>1亿的优质客户，问"近期有没有招行联系过"
    · 预警：3户以上重点客户反映"有招行联系过"，上报分行公司部负责人
- 完成时点：XXXX年XX月XX日前

━━━━━━━ 【三、主动作：研究准备】━━━━━━━━━━

▸ 研究准备（XX月XX日前）：华夏存款增速趋势判断

- 搞清楚这个问题是为了：为下月判断"华夏存款增速是否为趋势性变化"提供依据
- 研究责任人：企金部门（注：企金部/战略研究部均可承接，视各行实际）[6源一致确认]
- 研究内容（3项选2项即可，不用全做）：
    1. 华夏3年/5年期定存利率与我行差值（判断是否价格驱动）
    2. 近三月月均新增物理网点数量（判断是否渠道扩张驱动）
    3. 近3年Q1同比增速历史对比（判断是否季节性）
- 产出形式：一页书面判断报告（3~5个要点）【打头第一条结论必须最核心】
- 接收方：分行公司部负责人（或分行长）
- 转动作条件（注语形式，压缩版）：
    · 若华夏3年定存利率高于我行15bp以上 **且** 月均新网点>5个 → 下月启动挖转排查
    · 若华夏月增连续2个月<500亿 → 下月归档，不再触发针对华夏的存款动作
    · 其余情况：结论留存，下月补充数据后再判断
- 预计完成时点：XXXX年XX月XX日前

━━━━━━━ 【四、观察项】━━━━━━━━━━

▷ 观察项A：外币存款流入是否持续

▷ 观察项B：定期存款占比是否加速上升
    执行人/部门：企金部门（注：若无专门研究支持，可由企金部产品经理兼任）[6源一致确认]
    关注：样本行中3家以上月增占比>2个百分点 → 升级为主动作
```

## 三、三档条件压缩版说明（V5 特有）

V5 将 V4 的完整三档条件段落压缩为2行注语：

**V4（三档完整段落）**：
```
转动作条件：
    若[触发事实]达到[判断门槛]，则下月启动[对应动作]
    若[触发事实]达到[判断门槛]，则下月启动[对应动作]
    若[触发事实]达到[判断门槛]，则下月启动[对应动作]
```

**V5（2行注语）**：
```
转动作条件（注语形式，压缩版）：
    · 若华夏3年定存利率高于我行15bp以上 且 月均新网点>5个 → 下月启动挖转排查
    · 若华夏月增连续2个月<500亿 → 下月归档，不再触发
    · 其余情况：结论留存，下月补充数据后再判断
```

**为什么保留2行而不是1行**：3种情况（触发/归档/待定）各需要一个出口，2行是最低可读的压缩边界。

## 四、双门槛弹性写法（V5 更新版）

### 4.1 双门槛结构（V5 当前推荐）

| 层级 | 门槛 | 目的 | 执行要求 |
|------|------|------|---------|
| 重点排查 | 参考阈值（各行可调）| 防挖转 | 必须联系，确认续做意向 |
| 快速扫描 | 参考阈值（各行可调）| 摸底异常 | 不必全部联系，重点是有无异常信号 |

### 4.2 门槛弹性标注（V5 更新）

```
验证指标：
    重点：联系5000万以上存款到期客户，确认续做意向
    扫描：排查1000万以上存款客户有无异常信号（不必全部联系）

【注】：5000万/1000万为参考阈值，各行可按存量客户数量和执行承载力上下浮动。
收到更多反馈后，模板据此调整参考值。
```
