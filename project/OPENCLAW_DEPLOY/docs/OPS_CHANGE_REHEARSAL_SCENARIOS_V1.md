## 一、场景分析框架

**风险等级定义：**
- 🔴 **高风险（No-touch Zone）**：改了极可能引发全站漂移，需要白天人工确认
- 🟡 **中风险（Controlled Zone）**：改了需要按剧本回归，但可以受控完成
- 🟢 **低风险（Safe Zone）**：局部改动，回归范围小

**改动类型分类：**
- Type-A：S0 句子改动（硬约束句）
- Type-B：S1 句子改动（高风险同步句）
- Type-C：S2/S3 句子改动（可局部调整）
- Type-D：模块标题/标签改动
- Type-E：段落结构/顺序改动
- Type-F：样式/格式改动（通常 Safe Zone）
