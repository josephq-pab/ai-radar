## 三、V7 修复策略

### 修复幅度：回退修复型（中等）

- 精准针对 F1~F6，不推翻已确认方向的积累
- 不追求"比 V6 更短"，追求"比 V6 更容易懂 + 更像行动稿"
- 篇幅约 950~1000 字（比 V6 的 850 字多 100~150 字，是必要的信息厚度回归，不是 V5 回归）

### 三类处理方式

| 类型 | 处理方式 |
|------|---------|
| 保留 | 企金归口 / 双门槛结构 / 三段式 / 报告打头结论 / 部门+角色两级 |
| 回退 | 三档条件恢复到 V4/V5 完整段落 / 执行版说明层 / 操作细节层 |
| 重做 | 主动作格式（动词打头+具体操作标准）/ 三档数值背景 / 研究内容完成标准 |

## 四、V7 与 V6 的关键差异

| 维度 | V6 | V7 |
|------|----|----|

| 数值来源 | 不说明 | 明确或标注"各行可调" |
