## 一、看板定位

**本包是什么：**
- 不是看板系统（无任务/状态追踪）
- 不是路线图（无迭代计划）
- 不是项目管理页（无工作分配）
- 是：**只读能力与边界集中展示模块**（在现有页面中嵌入）

**本包解决什么问题：**
- 已可用能力、冻结能力、阻断能力、明确不做事项分散在多个页面，没有集中口径
- 运营者和协作者不知道这套系统现在"能做什么/不能做什么/为什么不能做"
