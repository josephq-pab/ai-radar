## 三、analyst_sources.json 最小修改草案

> ⚠️ 以下为草案，不自动生效。需人工 review 确认后再执行。

### 草案 1：降级顾慧君/王锟

将 `active: true` 改为 `active: false` 或移除，确保不占用抓取资源：

```json
// analyst-deposit-006 顾慧君
{
  "id": "analyst-deposit-006",
  "name": "顾慧君",
  "active": false,      // 草案：从 true 改为 false（文章年份为 2021-2022，已无新文章）
  "notes": "星图金融研究院投资策略研究中心主任，覆盖存款利率、银行股和大类资产配置视角。profile页有20篇以上文章，但最新文章年份为2022年，建议降级为 observe。"
}

// analyst-deposit-008 王锟
{
  "id": "analyst-deposit-008",
  "name": "王锟",
  "active": false,      // 草案：从 true 改为 false（文章年份为 2018-2019，已无新文章）
  "notes": "星图金融研究院投资策略研究中心高级研究员，profile页文章最新为2019年，建议降级为 observe。"
}
```

### 草案 2：新增温彬/曾刚 的 profileUrl

当前温彬/曾刚有 seedUrls，但 profileUrl 指向的平台可能需要单独抓取。建议补充 profileUrl 或明确 seedUrls 的抓取优先级。

```json

// 草案：明确抓取优先级，确保优先抓最新文章

// analyst-loan-002 曾刚

  "https://www.shifd.net/yanjiu/detail/10302.html"
```

## 四、对后续有效轮次的影响分析

### 当前困境根因

1. **主力来源薛洪言/付一夫断更**：sif.suning.com 对这两个高贡献来源在 2025 年后无新文章
2. **温彬/曾刚从未被抓取**：两个 high priority 来源有可用链接但尚未尝试
3. **已抓取的低价值来源（顾慧君/王锟）**：文章极旧，持续占用抓取资源但不贡献可用文章
4. **新来源（连平/周茂华/娄飞鹏）**：有 2026 年新文章，但数量有限（5条），占比仍低

### 扩展抓取后预期

若优先抓取温彬/曾刚（high priority，从未抓取）：
- 预期增加高优先级维度（对公贷款/整体）的新文章
- 可能打破当前"内容替换但无净新增"的僵局
- 降低薛洪言的占比，改善 OI-03 问题

### 不扩展的影响

若不扩展新来源：
- 后续轮次将持续表现为"内容替换型刷新"（fetchedAt 变，但 raw 不增）
- 可用文章池将逐渐陈旧（薛洪言/付一夫的 2024 年文章距今 1-2 年）
- preflight 会持续触发 D-续，但人工判断后可能持续维持 D（不进入轮次）

## 五、本文档定位与边界

**本文档不包含：**
- 不修改评分逻辑 / top-k / Gate A
- 不做自动抓取调度
- 不设计来源治理平台
- 不做 confirmLevel 重构
- 不做正式月度复盘报告

**本文档仅用于：**
- 支撑人工决策（是否扩展来源配置）
- 指导下一步配置变更（如何修改 analyst_sources.json）
- 评估来源扩展对可用池的实际影响

---

## 六、变更生效条件

1. 人工 review 本文内容
2. 用户确认批准配置变更
3. 在 analyst_sources.json 上执行草案修改（不在本文档上生效）
4. 执行一次 fetch 验证新来源可抓取
5. 验证 preflight 是否能识别新的有效刷新
