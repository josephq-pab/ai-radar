## 二、V5 四段结构

```
━━━ 【报告标题】━━━━━━━━━━━━━━━━━━
月度同业存款/贷款分析
（数据截至 XXXX年XX月）

━━━ 【一、本月最重要变化】（≤3条，每条不超过3行）━━━━━━━━━━
[四字判断 + 核心数据 + 一句话意义]

1. 【中信存款攻势超预期】
    中信银行当月新增存款2401亿，增量居样本行第一，超出预期。
    若持续，可能分流我行重点客户存款。

2. 【华夏存款增速领先但待验证】
    华夏银行年增+6.3%全行业最高；但月增仅+558亿，趋势待验证。

3. 【平安外币存款流入加快】
    平安银行外币存款月增+121亿，外币资金流入提速。

━━━━━━━ 【二、原因判断与对我行意义】━━━━━━━━━━

针对变化1（中信攻势）：两会后政策落地，对公存款同步增加（置信度：中）。
若持续——分流我行重点客户。

针对变化2（华夏提速）：定期储蓄利率竞争力较强+网点扩张（置信度：低；样本仅3个月）。
下月增速是否回落是关键信号。

针对变化3（平安外币流入）：美联储降息预期下，客户锁汇意愿下降（置信度：低）。
外币存款分流效应需持续跟踪。

━━━━━━━ 【三、三条建议】━━━━━━━━━━

1. 【立即排查中信挖转风险】
    目的：确认中信存款增量是否来自我行转移
    建议责任人：企金部门（注：企金部/战略研究部均可承接，视各行实际）[6源一致确认]
    验证指标：
        重点：联系5000万以上存款到期客户，确认续做意向
        扫描：排查1000万以上客户有无异常信号
        预警阈值：单笔转投≥1亿元即报

2. 【启动华夏存款策略研究】
    目的：为下月判断华夏存款增速是否趋势性变化提供依据
    建议责任人：企金部门（注：企金部/战略研究部均可承接，视各行实际）[6源一致确认]

3. 【完成一次招行贷存款联动客户扫描】
    目的：招行贷款利率2.54%已是行业最低，本月建基准数据
    建议责任人：分行公司部 / 客户经理

━━━━━━━ 【四、两个观察项 ━━━━━━━━━━━━

▷ 观察项A：外币存款流入是否持续
    连续2个月>100亿且我行同期外币存款流失>10% → 升级为主动作

▷ 观察项B：定期存款占比是否加速上升
    样本行中3家以上月增占比>2个百分点 → 升级为主动作
```
